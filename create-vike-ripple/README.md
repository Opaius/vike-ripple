# create-vike-ripple

Scaffold a Vike + Ripple TS project.

## Usage

```bash
# With Tailwind CSS (default)
npx create-vike-ripple my-app

# With Panda CSS
npx create-vike-ripple my-app --style pandacss

# Without any CSS framework
npx create-vike-ripple my-app --style none
```

The scaffold includes:
- Vike for routing and SSR
- Ripple TS for reactive UI
- Layout, Head, SEO config via `vike-ripple`
- Panda CSS or Tailwind CSS integration

See the [main repo](https://github.com/Opaius/vike-ripple) for full documentation.
